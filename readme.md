# 使用方法
将unload.csv放置于软件相同目录下然后双击运行，或者将符合格式的csv拖到exe上打开
## 格式如下
| english | chinese |
| ------ | ------ |
| Lockpicking|开锁|
|Storm Atronach|风暴元素|

*注意需要包含english,chinese这个表头*

# 输出
默认为output.txt,编码为utf8-BOM
这个文件应该用于xTranslator的批量翻译功能
